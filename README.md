# Paper Forge

This repository is being shaped into a modular macOS document productivity platform.

The approved product direction is SwiftUI for the frontend, Swift for the main app runtime, and Python only for specialized processing helpers behind adapters.

See [docs/architecture/architecture-decisions.md](./docs/architecture/architecture-decisions.md) for the canonical architecture and stack decision record.
See [AGENTS.md](./AGENTS.md) for the operating guide, roadmap, module ranking, and packaging strategy.

Current prototype and packaging helpers live in `work/`. Generated deliverables live in `outputs/`.
